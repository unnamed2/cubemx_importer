# CubeMX 导入引脚到原理图

一键快速 CubeMX 导入引脚到原理图

## 功能说明

本插件用于快速将CubeMX生成的\*.ioc文件中的引脚定义快速导入到 [嘉立创EDA专业版](https://pro.lceda.cn/) 。

1. 点击[嘉立创EDA专业版](https://pro.lceda.cn/)原理图页面顶部的菜单中高级->Import CubeMX IOC->从CubeMX IOC导入

2. 选择要打开的\*.ioc文件

3. 完成导入
